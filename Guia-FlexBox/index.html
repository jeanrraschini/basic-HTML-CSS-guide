Flex Container

Flex Container é todo bloco que recebe elementos filhos aos quais se aplicarão as propriedades complementares ao display: flex. O Flex Container tem como base de orientação os eixos X e Y.

Flex Item

Flex Item é todo elemento afetado pelas propriedades definidas no elemento pai (no caso, o flex container). O Flex Item tem o valor da sua propriedade display "normalizada" como inline-block.
A disposição ou a "forma como os elementos ficarão organizados" são definidas no Flex Container, onde anteriormente definimos o valor da propriedade display: flex.
Os Flex items podem ser alinhados e até ordenados de várias maneiras sob efeito das propriedades de alinhamento e disposição compatíveis com o Flex sendo as mais utilizadas delas:

  • justify-content - Alinhamento horizontal
  • align-items - Alinhamento vertical
  • flex-direction - Direção do alinhamento (Podendo ser em Linha ou Coluna)

Valores default

Ao atribuir o valor display: flex a um elemento ele automaticamente agrupa todos os elementos filhos (agora Flex Items) no canto superior esquerdo, alinhados da esquerda para a direita na ordem em quem estão declarados dentro do container. Por ora vamos guardar que:
"O canto esquerdo é o início da minha orientação horizontal (eixo X) e a extremidade superior (o topo) é o ponto inicial da minha orientação vertical (eixo Y)" Vamos lembrar que esses são os nossos "starts" no conceito de flex. O Justify Content

Agora que já vimos que o display: flex e como ele age inicialmente nos flex-items vamos começar a entender como funcionam suas propriedades adicionais. A primeira delas que veremos é o justify-content
Esta propriedade refere-se à disposição dos itens no Eixo X ou seja, na organização horizontal dos itens.
O principais valores que podem ser atribídos à propriedade justify-content são:

  • flex-start - Como vimos anteriormente ele agrupa os flex-items no ponto inicial do eixo X (o canto esquerdo quem olha para a tela. Exemplo acima)
  • flex-end - Tem um comportamento contrário ao flex-start, logo, agrupa os flex-items no ponto final do eixo X (canto direito de quem olha para a tela) Os exemplos desta sessão ficarão em containers maiores APENAS para melhor visualização.
